# PUNCH NEON - 2D 賽博霓虹拳擊對決

這是一款精緻且極具打擊感的 2D 賽博朋克霓虹風格拳擊遊戲。
支援 **單人 AI 練習**、**本地雙人對戰** 以及 **線上 WebSocket 連線對決**！

---

## 🎮 遊戲特色

1. **三位體型各異的角色**：
   - **Titan (泰坦)**: 重量型，身材高大，血量高、攻擊強，但移動與跳躍較慢。
   - **Swift (迅捷)**: 敏捷型，身材瘦小，移動與跳躍極快，出招快，但血量較少。
   - **Ryder (雷德)**: 平衡型，身材適中，屬性均衡，適合各類型玩家。
2. **極致的視覺打擊感**：
   - CRT 螢幕弧度與掃描線效果 (Scanlines)。
   - 毛玻璃選單 UI (Glassmorphism)。
   - 粒子噴射特效、防守能量盾，以及受擊時的震屏 (Screen Shake) 效果。
3. **多人連線機制**：
   - 基於 Node.js 與 Socket.io 構建。
   - 房間制對戰，輸入 4 位房號即可與好友即時連線。
   - 針對動作遊戲特別優化的「攻擊方命中判定」與對手位置插值平滑，體驗順暢。

## 🕹️ 遊戲操作 (Controls)

| 動作 | 玩家 1 (本地/線上) | 玩家 2 (本地雙人) |
| :--- | :--- | :--- |
| **左右移動** | `A` / `D` | `←` / `→` 方向鍵 |
| **跳躍** | `W` | `↑` 方向鍵 |
| **下蹲** | `S` | `↓` 方向鍵 |
| **輕拳 (速)** | `J` | `1` (小鍵盤或主鍵盤) |
| **重拳 (強)** | `K` | `2` (小鍵盤或主鍵盤) |
| **防守 (擋)** | `L` | `3` (小鍵盤或主鍵盤) |

*防守提示：當對手出拳時，按住防守鍵可進入「防盾狀態」，防禦成功可抵消 90% 傷害並免疫硬直與擊退！*

## 🚀 本地開發與運行

### 1. 安裝依賴套件
在專案根目錄下，開啟終端機執行：
```bash
npm install
```

### 2. 啟動伺服器
```bash
npm start
```
伺服器將在 `http://localhost:3000` 運行。

### 3. 本地測試連線
在瀏覽器中開啟兩個視窗，網址皆輸入 `http://localhost:3000`：
- 在視窗 A 點選「線上連線對戰」，點擊「創建房間」，會產生 4 位房間代碼（例如 `7431`），接著選擇您的角色。
- 在視窗 B 點選「線上連線對戰」，輸入剛才的代碼 `7431` 並點擊「加入」，選擇您的角色。
- 對手齊聚後，遊戲將自動啟動！

## 🌐 部署至 Render 指南

Render 能夠非常方便地免費部署此專案：

1. 將本專案上傳至您的 GitHub 帳號。
2. 註冊並登入 [Render](https://render.com/)。
3. 在 Render Dashboard 中，點擊 **New +** 選擇 **Web Service**。
4. 連結您剛才上傳的 GitHub 專案。
5. 配置以下設定：
   - **Runtime**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `node server.js`
   - **Instance Type**: `Free`
6. 點擊 **Deploy Web Service**。

部署完成後，Render 會為您生成一個二級網域網址 (例如 `https://punch-neon.onrender.com`)，您與好友即可在該網址進行跨地域連線格鬥！
